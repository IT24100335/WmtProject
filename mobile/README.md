# Crave Bites Mobile

React Native + Expo front end for the Crave Bites full stack project.

## Run

1. Install dependencies in this folder.
1. Set `EXPO_PUBLIC_API_URL` to your hosted backend API, for example:

```env
EXPO_PUBLIC_API_URL=https://your-backend.onrender.com/api
```

1. Start the app:

```bash
npm run start
```

## Notes

- Customer login, registration, menu, cart, checkout, orders, and profile are included.
- Admin login, dashboard, menu management, orders, promotions, inventory, feedback, and users are included.
- Menu image upload uses multipart form data.
- Payment is simulated for assignment/demo purposes.
